# Kobe Bryant's 2009 NBA Finals Statistics Analysis

## Introduction

I created this project as the final submission for my Intro to Data Science course at Middlebury College. The goal of the project was to apply my data analysis and data visualization skills using R-Studio and compatible packages. In this analysis, I delve into Kobe Bryant's statistics during the 2009 NBA Finals series and try to report an trends found in his performance.

## About the Data

The dataset used in this project consists of Kobe Bryant's statistics from the 2009 NBA Finals series. It's important to note that the data is somewhat inconsistent due to certain shortcomings in the data collection process, which are further discussed in the project report.

## Objective

The main objective of this project was to practice data analysis and data visualization techniques using R-Studio and relevant packages. Through this analysis, I aimed to uncover insights into Kobe Bryant's performance during the 2009 NBA Finals.
